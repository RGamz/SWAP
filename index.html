  • 1. Где найти иконку для входа в личный кабинет?

    Для создания нового аккаунта или для входа в уже существующий аккаунт кликните по соответствующей иконке в правом верхнем углу

  • 2. Как войти в личный кабинет, восстановить пароль или создать новую учетную запись?

    Если у вас уже есть наш аккаунт, то на открывшейся странице введите адрес электронной почты и пароль от вашего личного кабинета. Если вы забыли пароль, то вы можете восстановить его кликнув по ссылке «Забыли пароль?». Для создания нового аккаунта кликните по ссылке «Нет учетной записи? Создайте ее здесь»

  • 3. Как зарегистрироваться на сайте?

    В открывшейся форме обязательно укажите ваши полные ФИО. Эти данные необходимы для оформления документов о покупке на ваше имя. Укажите адрес электронной почты, который будете использовать для входа и придумайте пароль для входа на нашем сайте. Поля «Организация» и «ИНН» нужно заполнить только, если вы являетесь юридическим лицом.
